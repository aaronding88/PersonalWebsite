Current Project: Thruster

Thruster started off as a practice game for me, and it's grown into a passion project of mine for the past many months. It is half learning experience, half professional project. It is a basic bullet dodge game that uses only one input (left click/touch), which thrusts the character towards the direction of your input. However, the thruster exhaust that is created isn't just your 1980 Chevy's exhaust; it is a high-density super-charged ionic discharge (Think Tesla! Just kidding). It can damage enemies, reflect projectiles, and mess up all kinds of bad-guy mechanics. So, the player must decide whether he/she wants to use the thrust defensively, or offensively.
